# 📱 ImHere

**Stand out in the crowd at large events and let your companions find you easily.**

ImHere is an Android app that turns your phone into a custom visual signal: **color blinking** or **horizontal scrolling text**. Perfect for concerts, festivals, protests, or any place where you need to be spotted quickly.

---

## ✨ Features

- **Blink mode**: choose from 8 colors and 3 speeds (plus a steady option).
- **Scroll text mode**: write a custom message that scrolls horizontally.
  - 8 text colors.
  - 3 scroll speeds.
  - Adjustable text size via slider.
  - If the text fits on the screen, it is shown centred (no scroll).
- **Simple navigation**: Main screen → Configuration → Execution.

---

## 🔧 Prerequisites (for installation from a PC)

To install ImHere directly from your computer (without generating an APK) you need:

- A **PC with Windows, macOS or Linux**.
- **Android Studio** installed (download from [developer.android.com/studio](https://developer.android.com/studio)).
- An **Android phone** with a USB cable (a Google Pixel or any device with Android 7.0 or higher is recommended).
- Basic terminal knowledge (only to enable USB debugging).

> **Note**: This method is ideal for developers or users who want to test the app without publishing it on Google Play. It does not require a developer account or any payment.

---

## 📲 Step‑by‑step installation (from PC to phone)

### 1️⃣ Prepare your Android phone

1. **Enable Developer Options**:
   - Go to `Settings` → `About phone` → `Build number`.
   - Tap **7 times in a row** until you see “You are now a developer!”.

2. **Enable USB debugging**:
   - Go to `Settings` → `System` → `Developer options`.
   - Find `USB debugging` and turn it on.
   - Connect your phone to the PC with a USB cable. If a permission message appears, accept it and check “Always allow”.
